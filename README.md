# The Example For using FusionAuth and Golang

Simple demo for using [FusionAuth](http://fusionauth.io/) with Golang.

## Installation

clone with git first

```bash
git clone https://github.com/krissnawat/fusionauth-go
```

## Usage

```go
go run main.go
```
